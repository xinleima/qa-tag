# qa-tag
> The frontend of qa-tag model.

## Requirement
- node >= 8.0
- npm >= 5.0

## Install
```shell
# install dependencies of project
$ npm install
```

## How to run
### Development
```shell
# run the node server
$ node server

# run the react frontend
$ npm run start
```

### Production
```shell
# build react frontend to static files
$ npm run build

# run the server
$ node server
```
